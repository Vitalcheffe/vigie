# Code of Conduct

This project follows the [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be kind, be constructive, be inclusive. Vigie exists to protect isolated elders during heatwaves — anyone contributing shares that mission. Behavior that discriminates against any group, or that compromises the safety of beneficiaries, will not be tolerated.
